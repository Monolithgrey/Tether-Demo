# Tether — AI-Powered Access Intelligence

Interactive value engineering platform for healthcare patient access optimization. Built to quantify the economic impact of unfilled slots, missed appointments, and lost referrals.

## Three Paths

- **Build Your Case** — Capacity-first modeling for prospective health systems
- **Prove Your Value** — ROI documentation for existing customers with expansion modeling
- **See Real Results** — Narrative case studies with documented outcomes

## Run Locally

```bash
npm install
npm run dev
```

## Deploy to GitHub Pages

```bash
npm run deploy
```
